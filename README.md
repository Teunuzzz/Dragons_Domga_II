# Dragon's Dogma II Companion — v2.1 Data Edition

GitHub Pages PWA voor Teun.

## Uploaden
Upload de inhoud van deze map naar `Teunuzzz/Dragons_Domga_II` op branch `main` en zet Pages op `main / root`.

## Inhoud
- OP-route per vocation, standaard Fighter
- Verwerkte Nederlandse databestanden in `data/`
- Kaart met markers en crosshair-kalibratie
- Checklist/voortgang in localStorage
- Bronlinks in de app

## Belangrijk
De data is eigen, samengevatte/verwerkte game-guide data met bronlinks. Exacte markerposities zijn kalibreerbaar en worden steeds beter zodra ankerpunten goed worden gezet.
