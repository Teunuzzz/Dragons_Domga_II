# Dragon's Dogma II Companion - Teun v1.0

GitHub Pages repository: `teunuzzz.github.io/Dragons_Domga_II`

## Uploaden
1. Pak deze ZIP uit.
2. Upload alle bestanden naar de root van repository `Dragons_Domga_II`.
3. GitHub: Settings -> Pages -> Deploy from branch -> `main` -> `/root`.
4. App: `https://teunuzzz.github.io/Dragons_Domga_II/`
5. iPhone: open de link -> delen -> Zet op beginscherm.

## Wat zit erin
- Twee kaarttabbladen: OP-route en 100% completion.
- Vocation selector, standaard Fighter.
- Spoilers aan.
- Interactieve kaart met zoom/sleep.
- Filters per type marker.
- Zoekfunctie.
- Checklist met lokale opslag.
- Database-tab voor gear/quests.
- Marker-editor: klik op de kaart, vul naam/type in, opslaan.
- Offline PWA via service worker.

## Belangrijk
De eerste dataset is een eigen samengestelde starterdatabase met hoofdlocaties, OP-route en checklistclusters. Niet alle markerposities zijn officiële of pixel-perfect locaties. Met de marker-editor kun je tijdens het spelen markers corrigeren en exporteren.

Bronbasis voor structuur: Capcom official vocations, openbare wiki/gidsinformatie en jouw geüploade kaartafbeelding.
